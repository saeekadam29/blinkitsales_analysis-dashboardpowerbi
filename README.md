# blinkitsales_analysis-dashboardpowerbi
Power BI dashboard analyzing sales performance across outlets types and item categories.
Project presents an interactive Sales Performance Dashboard built using Power BI to analyze Blinkit's retail sales data.

The dashboard provides insights into:
- Sales trends over time
- Outlet performance comparison
- Product category contribution
- Item type analysis
- Customer rating patterns
The objective of this project is to derive actionable business insights using data visualization and DAX-driven metrics.
